预定义标识符

发表于2024-04-23|更新于2024-04-23|golang
|阅读量:

预定义标识符列表
append bool byte cap close complex
complex64 complex128 uint16 copy false float32
float64 imag int int8 int16 int32
uint32 int64 uint64 iota len make
new nil panic print println real
recover string true uint unit8 uintptr

这里面涵盖了类型，基础函数，关键字等。

append是切片的扩展函数。

cap是容量函数。

len是长度函数。

make是特殊变量声明并初始化函数。

new是声明变量函数。

panic是异常触发函数。

recover是恢复函数。

nil是空状态标识符。

其他基本都是类型标识符了。
